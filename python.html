Introdução

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991. Atualmente, possui um modelo de desenvolvimento comunitário, aberto e gerenciado pela organização sem fins lucrativos Python Software Foundation. Apesar de várias partes da linguagem possuírem padrões e especificações formais, a linguagem, como um todo, não é formalmente especificada. O padrão na pratica é a implementação CPython.

Python ganhou rapidamente popularidade desde os anos 2000 devido à sua simplicidade, legibilidade e uma vasta biblioteca de módulos. A linguagem encontrou uso em várias áreas, como desenvolvimento web, ciência de dados, aprendizado de máquina e automação.

Instalação e Configuração

Bom, a primeira coisa para instalar a linguagem de programação Python é acessar o site oficial de Python. Lá, você encontrará todas as informações necessárias para iniciar o processo de instalação.


Instalar e configurar programas em computadores pode parecer uma tarefa complicada, mas na realidade, é mais simples do que você imagina. Primeiro, você deve possuir o VSCode em seu dispositivo. Para instalar acesse Visual Studio Code.


Passo 1: Abra o Visual Studio Code: Após a instalação do VS Code, abra o aplicativo.

Passo 2: Clique na seção "Extensões" na barra lateral esquerda (ícone de quatro quadrados empilhados ou pressione Ctrl+Shift+X).

Passo 3: Na barra de pesquisa, digite "Python" e pressione Enter.

Passo 4: Clique no resultado "Python" e, em seguida, clique no botão "Instalar" na página da extensão.

Passo 5: Escreva e execute código: Agora, você pode escrever seu código Python no VS Code e executálo pressionando Ctrl+Enter ou clicando com o botão direito do mouse e selecionando "Executar Python File in Terminal".


Imprimindo o primeiro Hello World:

Para imprimir "Hello, World!" em Python, você pode usar a função print(). Aqui está um exemplo simples: print("Hello, World!") Basta copiar e colar esse código em um arquivo Python (.py) e executá-lo para ver a saída "Hello, World!" no console.

Sintaxe e Funções de Python

Além de tantas vantagens, que tornam Python a linguagem mais utilizada atualmente, também possui um diferencial na sintaxe e nas funções, o que torna o ato de "codar" muito mais prático, fácil e rápido.

Veja abaixo alguns exemplos de sintaxe com Python:


Comentários: Comentários em Python começam com o símbolo #. Eles são ignorados pelo interpretador e servem para documentar o código.

#isso eh um comentario

Estruturas Condicionais: Python usa if, elif (abreviação de "else if") e else para criar estruturas condicionais.

if idade < 18:
print("Menor de idade")
elif idade >= 18 and idade < 65:
print("Adulto")
else:
print("Idoso")

Para definir uma função em Python, você usa a palavra-chave def, seguida pelo nome da função, parênteses (), e um :. O corpo da função é indentado com espaços ou tabulação.

def saudacao(nome):
print("Olá, " + nome + "!")

Funções podem receber argumentos (valores passados quando a função é chamada) e esses argumentos podem ser usados como parâmetros (variáveis) dentro da função.

def soma(a, b):
resultado = a + b
return resultado
