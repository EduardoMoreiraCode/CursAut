Golang

Introdução

Go, ou Golang, é uma linguagem de programação de código aberto do Google, destacada por sua eficiência, concorrência e sintaxe clara. Projetada para desempenho rápido e desenvolvimento escalável, é uma escolha popular para sistemas e serviços de rede devido à sua velocidade de compilação e gerenciamento de memória automático.

Sintaxe e Funções

Você pode criar variáveis para armazenar informações, como números inteiros, decimais e palavras. Para declarar uma variável, usamos a palavra-chave "var" seguida do nome da variável e do tipo.

Exemplo:

var idade int

Declara uma variável chamada "idade" do tipo inteiro

nome := "João"

Declara e inicializa uma variável chamada "nome" com o valor "João"


Existem também as estruturas de controle, que nos permitem tomar decisões e repetir ações com base em condições. A seguir, alguns exemplos, que incluem estruturas "if/else" e "for".

Estrutura IF:

if idade >= 18 {
fmt.Println("Você é maior de idade.")
} else {
fmt.Println("Você é menor de idade.")
}

Estrutura de Laços FOR:

for i := 1; i <= 5; i++ {
fmt.Println(i)
}
